This is the diagonal size that a secondary display would need in order to have the same pixel density as the primary display. It is calculated as:
Secondary Display Size (S2)=H2H1⋅W1sin⁡(tan⁡−1(H2V2))\textrm{Secondary Display Size}~(S_2) = \frac{H_2}{H_1} \cdot \frac{W_1}{\Sin{\Arctan{\dfrac{H_2}{V_2}}}}Secondary Display Size (S2​)=H1​H2​​⋅sin(tan−1(V2​H2​​))W1​​
where:
  • W1 is the physical width of the display panel
  • H1 is the horizontal resolution of the primary display
  • H2 and V2 are the horizontal and vertical resolution of the secondary display
For a HRES × VRES secondary display with the same pixel density as a SIZE UNIT HRES × VRES primary display, the diagonal size of the secondary display is:
S2 = (W1H2) / (H1sin(tan−1(H2/V2)))
The derivation of this formula starts from the fact that the pixel densities of both displays are the same:
D1=D2D_1 = D_2D1​=D2​
Since pixel density D is equal to the horizontal resolution divided by the panel width (H ÷ W), this means:
H1W1=H2W2\frac{H_1}{W_1} = \frac{H_2}{W_2}W1​H1​​=W2​H2​​
If the equation for width is substituted in for W2:
H1W1=H2S2⋅sin⁡(tan⁡−1(H2V2))S2=H2H1⋅W1sin⁡(tan⁡−1(H2V2))\begin{aligned}\frac{H_1}{W_1} &= \frac{H_2}{S_2 \cdot \Sin{\Arctan{\dfrac{H_2}{V_2}}}}\\[10mm] S_2 &= \frac{H_2}{H_1} \cdot \frac{W_1}{\Sin{\Arctan{\dfrac{H_2}{V_2}}}}\end{aligned}W1​H1​​S2​​=S2​⋅sin(tan−1(V2​H2​​))H2​​=H1​H2​​⋅sin(tan−1(V2​H2​​))W1​​​
The equation for the width of the first display W1 can also be substituted in if an equation that is only in terms of diagonal size and resolution of the displays is desired:
S2=H2S1⋅sin⁡(tan⁡−1(H1V1))H1⋅sin⁡(tan⁡−1(H2V2))\begin{aligned}S_2 = \frac{H_2 S_1\cdot\Sin{\Arctan{\dfrac{H_1}{V_1}}}}{H_1 \cdot\Sin{\Arctan{\dfrac{H_2}{V_2}}}}\end{aligned}S2​=H1​⋅sin(tan−1(V2​H2​​))H2​S1​⋅sin(tan−1(V1​H1​​))​​
Or, if the Pythagorean form is preferred:
S2=H2S1(V2H2)2+1H1(V1H1)2+1\begin{aligned}S_2 = \frac{H_2 S_1 \sqrt{\left( \dfrac{V_2}{H_2} \right)^2 + 1}}{H_1 \sqrt{\left( \dfrac{V_1}{H_1} \right)^2 + 1}}\end{aligned}S2​=H1​(H1​V1​​)2+1​H2​S1​(H2​V2​​)2+1​​​
